# SHADOW × CHEATS - PANELS

A marketplace platform for managing products with owner-controlled pricing and product management.

## Features

✨ **User Features:**
- User registration and login system
- Browse available products
- Add products to shopping cart
- Direct purchase functionality
- View scanner information for each product

👤 **Admin/Owner Features:**
- Admin dashboard accessible only to owner
- Add new products
- Edit product names, prices, and scanners
- Delete products
- Full product management control

🎨 **Design:**
- Crimson red color scheme
- Dark theme with glowing effects
- Responsive design for mobile and desktop
- Smooth animations and transitions

## How to Use

### For Users:
1. Sign up with your email and password
2. Login to your account
3. Browse the PANELS section
4. View product details, prices, and scanner information
5. Add items to cart or buy directly
6. Proceed to checkout

### For Owner:
1. Login with owner email and password
2. Click the "⚙️ Admin" button in the navbar
3. Add new products with name, description, price, and scanner
4. Edit existing products (name, price, scanner)
5. Delete products as needed
